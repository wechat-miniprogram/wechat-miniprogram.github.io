# @mpflow/plugin-babel

Mpflow 的 babel 支持插件

# 说明

该插件用于给 Mpflow 的构架提供 babel.js 的集成，用于将 ESNext 代码编译到 ES5

提示

该插件通常为默认安装，不需要手动安装

# 安装

mpflow add babel
# 或
mpflow add @mpflow/plugin-babel
1
2
3

# 使用

直接通过 mpflow build 即可使用
