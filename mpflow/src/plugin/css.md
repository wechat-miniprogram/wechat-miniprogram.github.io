# @mpflow/plugin-css

> Mpflow 的 `css`, `less`, `sass`, `stylus` 等支持插件

## 说明

该插件用于给 Mpflow 的构架提供 `css`, `less`, `sass`, `stylus` 等功能的集成，可以使用 `.css`, `.less`, `.scss`, `.sass`, `.stylus` 文件代替 `.wxss` 文件

## 安装

```bash
mpflow add css
# 或
mpflow add @mpflow/plugin-css
```

## 使用

直接通过 `mpflow build` 即可使用
